Technical Gates

Session 1 | The Web Blueprint

By : Ameen Mohamed

Welcome & Introduction 15 mins

  • Welcoming trainees and defining the core aspects of the Front-End track.
  • Brief conceptual overview of what will be achieved by the end of the 6-day bootcamp.
  • Providing a full-picture insight into the final Portfolio project requirements.

Client-Server Architecture 30 mins

  • Detailed theoretical explanation of Client-Server architecture.
  • Analyzing the lifecycle of a Request from the browser to the Server, and the subsequent Response.
  • Defining the roles of HTML (Structure), CSS (Styling), and JavaScript (Logic).

Environment Setup 10 mins

  • Guiding trainees to open the code editor (VS Code).
  • Installing fundamental extensions like Live Server and code formatters like Prettier.

Initial Application & Boilerplate 15 mins

  • Creating a new project folder and the first index.html file.
  • Explaining the default lines of the HTML5 Boilerplate line by line (DOCTYPE, html tag, head metadata, title, and body).

Text Structure & Typography Basics 30 mins

  • Introduction to essential text tags: Headings (h1 to h6). Emphasis on not exceeding one h1 per page for SEO constraints.
  • Understanding paragraphs (p) and semantic text formatting tags like strong and em vs standard tags.

Lists Structure 10 mins

  • Unordered Lists (ul), Ordered Lists (ol), and List Items (li).
  • Explaining the essential role of lists in building Navigation Bars to prepare them for practical implementation later.

In-Class Task: Personal Article 20 mins

  • Trainees are challenged to build a structure for a personal article.
  • Requirements: Main heading, two paragraphs introducing themselves, and a bulleted list of hobbies.
  • The instructor walks around to ensure practical engagement and live error correction.

Rest Break 15 mins

  • A short break for trainees to refresh their energy and maintain concentration metrics.

Hyperlinks Mastery 25 mins

  • Analyzing the Anchor tag (a) and its dedicated attributes, primarily href (routing path).
  • Explaining the target attribute concept with various values (especially _blank for new tabs).
  • Navigating logically between internal website pages versus external destinations.

Images and Directing Paths 20 mins

  • Exploring the Image tag (img) and the critical necessity of the alt attribute for Accessibility and SEO algorithms.
  • Crucial Lesson: Thoroughly contrasting Absolute Paths versus Relative Paths.
  • Mastering directory traversal using double dots (..) to avoid common asset loading issues.

Live Demo: Media Integration 10 mins

  • Real-time demonstration of adding a personal portrait and linking social media accounts within the previously authored miniature article document.

Tabular Data (Tables) 20 mins

  • Building basic tables (table, tr, td, th syntax).
  • Providing historical context about table-based web design, and demonstrating how modern usage is restricted solely to displaying relational data sets.

Interactive Forms and Methods 30 mins

  • Entering the data input realm. Discussing the form tag structure.
  • Elucidating the fundamental theoretical differences between GET and POST payload transmissions alongside the action attribute.

Deep Dive into Inputs 20 mins

  • Extensive exploration of input field types: text, email, password, numbers, checkboxes, radial buttons, and submit handlers.
  • Demonstrating programmatic linking natively using the label tag associated with corresponding ID hooks to enhance UX.

Rest Break 15 mins

  • Mental rest block before venturing into the overarching project architectural skeleton.

Semantic Web Paradigms 30 mins

  • Transitioning away from arbitrary division tags (div, span) whilst clarifying block-level vs inline-level behaviors.
  • Introduction to semantic layout constraints (header, nav, section, article, aside, footer).
  • Emphasizing that these tags do not alter visual dimensions drastically but drastically improve algorithmic parsing readability.

Live Demo: Structuring the Portfolio 25 mins

  • Setting the actual project foundation. The instructor types out the holistic layout scope.
  • Authoring the structural Header holding the logo identifier and Navigation links.
  • Engineering the textual and visual representation of the Hero introductory section.

Live Demo: Completing the Skeleton 30 mins

  • Continuing the authoring procedure to encapsulate the About Me and Skills sections.
  • Utilizing previously acquired knowledge around unordered lists to formulate a skill matrix block.

Final In-Class Project Challenge 40 mins

  • The critical task of the day. Trainees must self-engineer the concluding Sections of their subjective Portfolio.
  • Scope entails formulating the Projects exhibition structure (to act as a Grid ecosystem later).
  • Constructing the Contact Me section dynamically dependent entirely on comprehensive Form inputs parameters.

Conclusion & Wrap-up 10 mins

  • Summarizing constructed assets and systematically reviewing the document hierarchy top-down.
  • A quick psychological preparation briefing regarding the CSS pipeline on day two and how rigid syntax will transition into vivid aesthetics.
